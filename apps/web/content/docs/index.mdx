**Emoteer** is a framework-agnostic toolkit for building rich emoji experiences — pickers, reactions, autocomplete, and emoji-aware inputs — with primitives that are headless by default and styled when you want them to be.

It ships a shared data and state core (`@emoteer/core`) plus thin framework bindings. React is production-ready today; Svelte and Vue are on the roadmap.

## Design principles

- **Headless first.** Compound primitives give you full control over markup, ARIA and styling. Defaults are opt-in.
- **Framework agnostic.** The core package ships data, search and state; bindings are a thin layer.
- **Typed end-to-end.** Every event, slot and prop is typed. No `any`s in public APIs.
- **Zero config.** Ship a full picker in ten lines. Customize only what you need.
